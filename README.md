# Portfolio Corporate

AI エージェントを活用して制作した WordPress 企業サイトテーマのデモです。

WordPress テーマ構造、ACF による入力項目、固定ページテンプレート、ローカル開発手順の検証を目的としています。

## 要件

- WordPress 6.x
- PHP 8.0+
- Advanced Custom Fields

## 主な実装

- フロントページ用テンプレート
- 固定ページテンプレート
- カスタム投稿タイプ `news`
- ACF ローカルフィールド登録
- SCSS ビルド設定
- ローカル検証用スクリプト

## セットアップ

1.  `wp-content/themes/` に配置する。
2. 管理画面でテーマを有効化する。
3. Advanced Custom Fields を有効化する。
4. 固定ページを作成し、表示設定でホームページに指定する。
5. パーマリンク設定を保存する。

## ACF

フロントページ用フィールドは `functions.php` で登録しています。管理画面での手動作成は不要です。

## 開発用ファイル

`tools/` はローカル開発用スクリプト。

ライセンスは各サービスの規約に従います。
